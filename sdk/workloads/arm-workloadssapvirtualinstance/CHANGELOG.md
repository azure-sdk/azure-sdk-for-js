### Breaking Changes

  - Removed operation group SAPApplicationServerInstancesOperations
  - Removed operation group SAPCentralServerInstancesOperations
  - Removed operation group SAPDatabaseInstancesOperations
  - Removed operation group SAPVirtualInstancesOperations
  - Class WorkloadsClient has a new signature
  - Interface SAPApplicationServerInstance no longer has parameter properties
  - Interface SAPCentralServerInstance no longer has parameter properties
  - Interface SAPDatabaseInstance no longer has parameter properties
  - Interface SAPVirtualInstance no longer has parameter properties
  - Type of parameter diskVolumeConfigurations of interface DiskConfiguration is changed from Record<string, DiskVolumeConfiguration> to {
        [propertyName: string]: DiskVolumeConfiguration;
    }
  - Type of parameter info of interface ErrorAdditionalInfo is changed from Record<string, any> to Record<string, unknown>
  - Type of parameter configurationType of interface FileShareConfiguration is changed from FileShareConfigurationType to "CreateAndMount" | "Mount" | "Skip"
  - Type of parameter deploymentType of interface InfrastructureConfiguration is changed from SAPDeploymentType to "SingleServer" | "ThreeTier"
  - Type of parameter osType of interface OSConfiguration is changed from OSType to "Linux" | "Windows"
  - Type of parameter configurationType of interface SAPConfiguration is changed from SAPConfigurationType to "Deployment" | "DeploymentWithOSConfig" | "Discovery"
  - Type of parameter volumeConfigurations of interface SAPDiskConfigurationsResult is changed from Record<string, SAPDiskConfiguration> to {
        [propertyName: string]: SAPDiskConfiguration;
    }
  - Type of parameter deploymentType of interface SAPSizingRecommendationResult is changed from SAPDeploymentType to "SingleServer" | "ThreeTier"
  - Type of parameter userAssignedIdentities of interface SAPVirtualInstanceIdentity is changed from Record<string, UserAssignedIdentity | null> to {
        [propertyName: string]: UserAssignedIdentity | null;
    }
  - Type of parameter namingPatternType of interface SingleServerCustomResourceNames is changed from NamingPatternType to "FullResourceName"
  - Type of parameter softwareInstallationType of interface SoftwareConfiguration is changed from SAPSoftwareInstallationType to "External" | "SAPInstallWithoutOSConfig" | "ServiceInitiated"
  - Type of parameter namingPatternType of interface ThreeTierCustomResourceNames is changed from NamingPatternType to "FullResourceName"
  - Type of parameter tags of interface TrackedResource is changed from Record<string, string> to {
        [propertyName: string]: string;
    }
  - Type of parameter tags of interface UpdateSAPApplicationInstanceRequest is changed from Record<string, string> to {
        [propertyName: string]: string;
    }
  - Type of parameter tags of interface UpdateSAPCentralInstanceRequest is changed from Record<string, string> to {
        [propertyName: string]: string;
    }
  - Type of parameter tags of interface UpdateSAPDatabaseInstanceRequest is changed from Record<string, string> to {
        [propertyName: string]: string;
    }
  - Type of parameter tags of interface UpdateSAPVirtualInstanceRequest is changed from Record<string, string> to {
        [propertyName: string]: string;
    }
  - Type of parameter dataDiskNames of interface VirtualMachineResourceNames is changed from Record<string, string[]> to {
        [propertyName: string]: string[];
    }
  - Class WorkloadsClient no longer has parameter pipeline
  - Removed Enum KnownVersions
  - Removed function restorePoller
